1. Prompt Engineering 
➢​ Nghiên cứu, thiết kế, và tối ưu hóa prompt cho các mô hình AI sinh (Generative AI).

2. Natural Language Processing (NLP) 
➢​ Hỗ trợ phát triển các ứng dụng xử lý ngôn ngữ tự nhiên (NLP) như phân tích ngữ

➢​ Áp dụng các mô hình AI để xây dựng giải pháp thực tế, đáp ứng nhu cầu khách hàng. 
➢​ Nghiên cứu, kiểm tra và tối ưu hóa hiệu suất của các mô hình NLP. 
3. Phát triển Web Application (WebApp) 
➢​ Hỗ trợ xây dựng và phát triển các ứng dụng web tích hợp công nghệ AI. 
➢​ Tham gia thiết kế giao diện người dùng (UI) và tích hợp các API AI vào ứng dụng 
web. 
➢​ Đảm bảo hiệu suất tối ưu và trải nghiệm người dùng xuất sắc. 
4. Lập trình Python 
➢​ Xây dựng và triển khai các công cụ AI/ML sử dụng Python. 
➢​ Tối ưu mã nguồn, cải thiện hiệu suất của các ứng dụng hiện tại. 
➢​ Làm việc với các thư viện Python như TensorFlow, PyTorch, spaCy, NLTK hoặc các 
framework như Flask, FastAPI.
